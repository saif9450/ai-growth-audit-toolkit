# AI Growth Audit Toolkit

An open-source toolkit designed to help startups, D2C brands, and operators run structured growth audits using AI.

---

## 🚨 Problem

Most early-stage companies struggle with:
- No clarity on why growth is stuck
- Poor funnel performance
- Inefficient ad spend
- Lack of structured analysis

High-quality consulting is expensive and inaccessible.

---

## 💡 Solution

This toolkit provides:
- AI-powered audit prompts
- Structured growth frameworks
- Execution-ready templates

---

## 🎯 Who This Is For

- Startup founders
- D2C operators
- Growth marketers
- Indie hackers

---

## 📦 What's Inside

- SEO audit prompts  
- Funnel analysis frameworks  
- Paid ads evaluation prompts  
- Notion-based audit system (coming)

---

## ⚙️ How to Use

1. Copy prompts into Claude
2. Input your business data
3. Follow structured audit output
4. Implement recommendations
